Building an Interactive View of the Iris Data Set

This page is a test bed, constructed for the purpose of comparing a static view of the Iris Data Set with an interactive one. The static view is constructed with Python's matplotlib library. The interactive view, embedded in this page, is contructed with D3. The first task is to visually inspect the data.
